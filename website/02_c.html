    C Programming Language Tutorial

    Introduction to C

    C is a general-purpose programming language that was developed in the early 1970s by Dennis Ritchie at Bell Laboratories. It is a high-level language that provides low-level access to computer memory through pointers, making it efficient and suitable for system-level programming.
    C is known for its simplicity, efficiency, and portability. It was designed to be a relatively small language with a minimal set of keywords and constructs, which makes it easy to learn and write code in. The language provides a structured approach to programming, allowing developers to break down programs into modules or functions.
    One of the key features of C is its ability to directly manipulate memory, which gives programmers precise control over the hardware. This feature makes C well-suited for developing operating systems, embedded systems, device drivers, and other system-level software.
    C is a procedural language, meaning that programs are organized into functions that manipulate data. It supports a wide range of data types, including integers, floating-point numbers, characters, and arrays. C also provides operators for performing arithmetic, logical, and bitwise operations on these data types.
    Another important aspect of C is its extensive standard library, which provides a collection of functions and macros that can be used to perform various operations, such as input/output, string manipulation, memory allocation, and mathematical computations.
    Learning C can be a valuable skill for aspiring programmers, as it provides a solid understanding of fundamental programming concepts and enhances problem-solving abilities. It's a language that emphasizes efficiency and control, making it suitable for various domains and giving developers a strong foundation for learning other programming languages.

    Structure of a C program

    The structure of a C program consists of various components that work together to define and execute the program. Here's a breakdown of the typical structure of a C program:
    1. Preprocessor Directives: These are statements that begin with a hash symbol (#) and are processed by the preprocessor before the actual compilation. They are used to include header files, define constants, and perform other preprocessing tasks.
    2. Header Files: These files contain function prototypes and definitions necessary for the program. They are included using the #include directive at the beginning of the program. Commonly used header files include stdio.h for input/output operations, stdlib.h for memory allocation, math.h for mathematical functions, etc.
    3. Global Declarations: This section is used to declare global variables and define constants that can be accessed throughout the program. Global variables have a scope that extends across multiple functions.
    4. Function Declarations: Function declarations specify the return type, name, and parameters (if any) of the functions used in the program. They provide a way to inform the compiler about the existence of functions before they are defined or used.
    5. Function: Every C program must have a main function, which serves as the entry point of the program. It is where the program execution begins and ends. The main function may accept command-line arguments (int argc, char* argv[]), although it is not required.
    6. Statements and Expressions: Inside the main function (or other user-defined functions), you write the actual statements and expressions that perform the desired operations of the program. These can include variable declarations, assignment statements, control flow statements (if-else, switch-case), loops (for, while, do-while), function calls, and other C language constructs.
    7. User-defined Functions: You can define your own functions in addition to the main function to break down the program's logic into smaller, manageable pieces. These functions can have their own local variables and perform specific tasks within the program. Function definitions typically appear after the main function or in separate files.
    8. Return Statements: The main function (and other functions if applicable) should end with a return statement, which specifies a value to be returned to the calling function or the operating system. In the main function, a return statement with no value (or return 0;) indicates successful program execution.

    Program

    
                #include < stdio.h >

                // Global variable declaration
                 int globalVariable = 10;
                
                // Function declaration
                int addNumbers(int a, int b);
                
                // Main function
                int main() {
                    // Local variable declaration
                    int x, y, sum;
                    
                    // Code statements
                    printf("Enter two numbers: ");
                    scanf("%d %d", &x, &y);
                    
                    // Function call
                    sum = addNumbers(x, y);
                    
                    // Output result
                    printf("Sum: %d\n", sum);
                    
                    return 0;
                }
                
                // Function definition
                int addNumbers(int a, int b) {
                    return a + b;
                }

    Data types in C

    In C programming, data types are used to define the type and size of data that variables can hold. Here are the commonly used data types in C:
    1. Integers
      • int: Used to store whole numbers (both positive and negative) within a specified range.
      • short: Similar to int, but typically has a smaller range and requires less memory.
      • long: Similar to int, but typically has a larger range and may require more memory.
      • unsigned int: Used to store only non-negative whole numbers (zero and positive values).
    2. Floating-point Numbers
      • float: Used to store single-precision floating-point numbers with decimal places.
      • double: Used to store double-precision floating-point numbers, providing higher precision than float.
    3. Characters
      • char: Used to store individual characters or small integers. It can represent ASCII characters and other character sets.
    4. Arrays
      • int[], float[], char[], etc.: Used to store the memory address of another variable. Pointers allow for dynamic memory allocation and efficient manipulation of memory.
    5. Structures
      • struct: sed to define a custom data type that can hold multiple variables of different data types grouped together.
    6. Enumerations
      • enum: Used to define a set of named constant values.
    7. Void
      • Void: Used to indicate the absence of a value or when a function does not return any value.
    8. Derived Types
      • Arrays: Used to store a collection of multiple values of the same data type in contiguous memory locations.
      • Pointers: Used to store memory addresses of variables or functions.
      • Structures: Used to define a custom data type that can hold multiple variables of different data types grouped together.
      • Unions: Similar to structures, but different variables within a union share the same memory space.
    It's important to note that C is a statically-typed language, which means that variables must be declared with their specific data types before they can be used. Different data types have different ranges and memory requirements, so it's essential to choose the appropriate data type based on the requirements of your program to ensure efficient memory usage and accurate representation of data.
    Additionally, C allows for user-defined data types through structures and enumerations, which can help in organizing and manipulating complex data structures.

    Variables and Constants

    In C programming, variables and constants are used to store and represent data. They play a crucial role in storing values that can be manipulated and utilized within a program. Here's an explanation of variables and constants in C
    1. Variables
      • Variables are named memory locations that can hold values of a specific data type.
      • They allow programmers to store and manipulate data during program execution.
      • To use a variable, it must be declared before it is used, specifying its data type and optionally assigning an initial value.
      • Variables can have different scopes, such as local variables within a function or global variables accessible throughout the program.
      • The value of a variable can be changed during program execution by assigning new values to it.

      Program

      
        #include 
        int main() {
            // Variable declaration
            int num1;
            int num2 = 5; // Variable declaration with initialization
        
            // Variable assignment
            num1 = 10;
        
            // Variable usage
            int sum = num1 + num2;
            printf("Sum: %d\n", sum);
        
            return 0;
        }
    2. Constants
      • Constants are fixed values that cannot be changed during program execution.
      • They are used to represent values that remain constant throughout the program.
      • Constants can be of different data types, such as integer constants, floating-point constants, character constants, or string constants.
      • In C, constants are typically defined using the const keyword or by using preprocessor directives such as #define.
      • Constants provide meaningful names to values that are used repeatedly in the program, enhancing code readability and maintainability.

      Program

      
        #include 
        #define PI 3.14159    // Preprocessor directive to define constant
        const int MAX_VALUE = 100;   // Declaration of constant variable using 'const'
        
        int main() {
            // Usage of constants
            float radius = 5.0;
            float circumference = 2 * PI * radius;
            printf("Circumference: %f\n", circumference);
        
            int array[MAX_VALUE];
            // ...
        
            return 0;
        }

    Operators

    In C programming, operators are symbols or keywords that perform operations on one or more operands (variables, constants, or expressions) and produce a result. They allow for arithmetic computations, logical evaluations, assignment of values, and more. Here are the different types of operators in C:
    1. Arithmetic Operators:
      • +: (Addition): Adds two operands.
      • -: (Subtraction): Subtracts the second operand from the first.
      • *: (Multiplication): Multiplies two operands.
      • /: (Division): Divides the first operand by the second.
      • %: (Modulus): Returns the remainder of the division operation.
      • ++: (Increment): Increments the value of the operand by 1.
      • --: (Decrement): Decrements the value of the operand by 1.
    2. Relational Operators
      • ==: (Equal to): Checks if two operands are equal.
      • !=: (Not equal to): Checks if two operands are not equal.
      • <: (Less than): Checks if the first operand is less than the second.
      • >: (Greater than): Checks if the first operand is greater than the second.
      • <=: (Less than or equal to): Checks if the first operand is less than or equal to the second.
      • >=: (Greater than or equal to): Checks if the first operand is greater than or equal to the second.
    3. Logical Operators
      • &&: (Logical AND): Returns true if both operands are true.
      • ||: (Logical OR): Returns true if at least one of the operands is true.
      • !: (Logical NOT): Negates the value of the operand.
    4. Assignment Operators
      • = : (Simple Assignment): Assigns the value on the right to the variable on the left.
      • +=, -=, *=, /=, %= : (Compound Assignment): Performs the respective arithmetic operation and assigns the result to the variable.
      • <<=, >>=, &=, |=, ^= : (Bitwise Assignment): Performs the respective bitwise operation and assigns the result to the variable.
    5. Bitwise Operators
      • &: (Bitwise AND): Performs bitwise AND operation on the binary representation of the operands.
      • |: (Bitwise OR): Performs bitwise OR operation on the binary representation of the operands.
      • ^: (Bitwise XOR): Performs bitwise exclusive OR operation on the binary representation of the operands.
      • <<: (Left Shift): Shifts the bits of the first operand to the left by the number of positions specified by the second operand.
      • >>: (Right Shift): Shifts the bits of the first operand to the right by the number of positions specified by the second operand.
      • ~: (Bitwise NOT): Flips the bits of the operand.
    6. Conditional Operator
      • ?: : (Conditional Operator or Ternary Operator): Evaluates a condition and returns one of two values based on the result of the condition.

    Control Flow

    In C programming, control flow refers to the order in which statements and instructions are executed within a program. Control flow allows you to make decisions, loop over code blocks, and control the flow of execution based on certain conditions. Here are the key control flow structures in C
    1. Conditional Statements
      • if statement: Executes a block of code if a given condition is true. It can be followed by an optional else statement for an alternative block of code to execute if the condition is false.

        Example

        
        if (condition) {
            // Code to execute if condition is true
        } else {
            // Code to execute if condition is false
        }
      • switch statement: Evaluates an expression and performs different actions based on its value. It consists of multiple case labels and an optional default label.

        Example

        
        switch (expression) {
            case value1:
                // Code to execute if expression matches value1
                break;
            case value2:
                // Code to execute if expression matches value2
                break;
            // more case statements
            default:
                // Code to execute if no case matches expression
        }
    2. Looping Statements
      • for loop: Repeats a block of code a fixed number of times, using a control variable and an initial condition, an iteration condition, and an update statement.

        Example

        
                        for (initialization; condition; update) {
                            // Code to repeat
                        }
      • while loop: Repeats a block of code as long as a given condition is true.

        Example

        
                        while (condition) {
                            // Code to repeat
                        }
      • do-while loop: Repeats a block of code at least once, and then continues repeating as long as a given condition is true.

        Example

        
                        do {
                            // Code to repeat
                        } while (condition);

    3. Jump Statements

      • break statement: Terminates the execution of a loop or switch statement, and transfers control to the next statement after the loop or switch block.
      • continue statement: Skips the remaining code within a loop and proceeds to the next iteration.
      • goto statement: Transfers control to a labeled statement within the same function.

    4. Function Calls

      • Functions allow you to encapsulate a block of code that can be called and executed at different points in the program.
      • Function calls allow you to transfer control to the function, execute its code, and return back to the calling point.

    Arrays in C

    In C programming, an array is a collection of elements of the same data type stored in contiguous memory locations. Arrays provide a convenient way to store and manipulate multiple values under a single variable name. Here's an overview of arrays in C
    1. Declaration and Initialization
      • To declare an array, you specify the data type of its elements and the number of elements it can hold. The syntax is as follows:
        • 
        data_type array_name[size];

        Example

        
        int numbers[5];  // Declaration of an integer array with size 5
      • You can also initialize an array at the time of declaration by specifying the values inside braces {}
        • 
        data_type array_name[size] = {value1, value2, ..., valueN};

        Example

        
        int numbers[5] = {1, 2, 3, 4, 5};  // Initialization of an integer array
    2. Accessing Array Elements
      • Array elements are accessed using zero-based indexing. The first element has an index of 0, the second element has an index of 1, and so on.
        • 
                    array_name[index]

        Example

        
                    int number = numbers[2];  // Accessing the third element of the 'numbers' array
    3. Modifying Array Elements
      • Array elements can be modified by assigning a new value to the desired element using the assignment operator =.
        • 
                    array_name[index] = new_value;

        Example

        
                    numbers[3] = 10;  // Modifying the fourth element of the 'numbers' array
    4. Array size
      • The size of an array is determined at the time of declaration and cannot be changed during program execution.
      • The number of elements an array can hold is specified in the square brackets [].
      • The size of an array can be accessed using the sizeof operator.
        • 
                    sizeof(array_name) / sizeof(array_name[0])

        Example

        
                    int size = sizeof(numbers) / sizeof(numbers[0]);  // Calculating the size of the 'numbers' array
    5. Iterating Over an Array:
      • Looping structures such as for or while can be used to iterate over the elements of an array using index values. 
        • 
                    for (int i = 0; i < array_size; i++) {
                        // Access and process array elements using 'array_name[i]'
                    }

        Example

        
                    for (int i = 0; i < size; i++) {
                        printf("%d ", numbers[i]);  // Printing elements of the 'numbers' array
                    }

    Fucntion in C

    In C programming, functions are a fundamental building block used to organize and modularize code. A function is a self-contained block of code that performs a specific task and can be called from different parts of a program. Here's an overview of functions in C
    1. Function Declaration and Definition
      • A function declaration tells the compiler about the function's name, return type, and the types of its parameters (if any).
        • 
        return_type function_name(parameter_list);

        Example

        
        int sum(int num1, int num2);  // Function declaration
      • A function definition provides the actual implementation of the function. It includes the function header (return type, name, and parameters in parentheses) and a body containing the code to be executed.
        • 
        return_type function_name(parameter_list) {
            // Function body (code to be executed)
        }

        Example

        
        int sum(int num1, int num2) {  // Function definition
            int result = num1 + num2;
            return result;
        }
    2. Function Calling
      • To use a function, you call it by its name followed by parentheses, passing arguments (if any) that match the parameter list. 
        • 
                        return_type variable = function_name(argument_list);

        Example

        
                        int a = 5, b = 3;
int result = sum(a, b);  // Function call
    3. Function Return
      • A function can optionally return a value using the return statement. The return type in the function declaration specifies the type of value to be returned. If the return type is void, the function does not return a value.
        • 
                        return_type function_name(parameter_list) {
                            // Code
                            return value;  // Return statement
                        }

        Example

        
                        int sum(int num1, int num2) {
                            int result = num1 + num2;
                            return result;  // Return statement
                        }
    4. Function Parameters
      • Function parameters allow you to pass values from the calling code to the function. Parameters are defined in the function declaration and used in the function definition.
        • 
                        return_type function_name(parameter_type parameter_name) {
                            // Code using the parameter
                        }

        Example

        int square(int num) {
                        int result = num * num;
                        return result;
                    }
      • Multiple parameters can be separated by commas in both the function declaration and definition.
    5. Function Prototypes
      • A function prototype is a forward declaration of a function. It provides the function's signature (name, return type, and parameter list) without providing its implementation.
      • Prototypes are typically placed at the beginning of the program or in header files to allow functions to be called before they are defined.
      • Prototypes are optional if the function is defined before it is called in the program. 
        • 
                        return_type function_name(parameter_list);

        Example

        
                        int sum(int num1, int num2);  // Function prototype

    Recursion in C

    Recursion is a programming technique where a function calls itself to solve a problem by breaking it down into smaller, simpler instances. It is a powerful concept used in various algorithms and data structures.
    Here are some key points about recursion:
    1. Base Case: Recursion involves defining a base case that represents the simplest form of the problem. It acts as the terminating condition for the recursive function and helps prevent infinite recursion.
    2. Recursive Case: The recursive function calls itself, typically with a modified set of parameters or input, to solve a smaller version of the original problem. The function continues to call itself until it reaches the base case.
    3. Recursion vs. Iteration: Recursion can often be used as an alternative to iteration (looping). Both approaches can solve the same problems, but recursion tends to be more concise for certain types of algorithms, such as those based on tree traversal or divide-and-conquer.
    4. Infinite Recursion: Without a proper base case or termination condition, recursion can lead to infinite recursion, causing the program to run indefinitely and potentially resulting in a stack overflow error.

      5. Example

      
        #include 

        // Recursive function to calculate the factorial of a number
        unsigned int factorial(unsigned int n)
        {
            // Base case: factorial of 0 is 1
            if (n == 0) {
                return 1;
            }
            // Recursive case: calculate factorial using recursion
            else {
                return n * factorial(n - 1);
            }
        }
        
        int main()
        {
            unsigned int num = 5;
            unsigned int result = factorial(num);
            
            printf("Factorial of %u is %u\n", num, result);
            
            return 0;
        }

    Pointer in C

    Pointers are a powerful feature in C programming that allow you to directly manipulate memory addresses and access data indirectly. They provide flexibility and efficiency in memory management and are widely used in various programming tasks. Here's an overview of pointers in C:
    1. Pointer Declaration and Initialization
      • A pointer is a variable that stores the memory address of another variable. It is declared by using the * symbol in the declaration. 
        • 
        data_type *pointer_name;

        Example

        
        int *ptr;  // Declaration of an integer pointer
      • Pointers can be initialized with the address of a variable using the address-of operator &.
        • 
        pointer_name = &variable_name;

        Example

        
        int number = 10;
        int *ptr = &number;  // Initialization of the pointer with the address of 'num'
    2. Pointer Dereferencing:
      • Dereferencing a pointer allows you to access the value stored at the memory address it points to. It is done using the dereference operator *. 
        • 
        *pointer_name;

        Example

        
        int value = *ptr;  // Dereferencing the pointer to access the value it points to
    3. Pointer Arithmetic
      • Pointers can be manipulated using arithmetic operations. Adding an integer to a pointer advances it by the specified number of elements, based on the size of the data type it points to. 
        • 
        pointer_name = pointer_name + value;

        Example

        
        int *ptr = /* some address */;
        ptr = ptr + 1;  // Advances the pointer by one integer size
    4. Null Pointer
      • A null pointer is a pointer that does not point to any valid memory address. It is typically used to indicate the absence of a meaningful value. 
        • 
        data_type *pointer_name = NULL;

        Example

        
        int *ptr = NULL;  // Declaration and initialization of a null pointer
    5. Pointer to Pointer
      • Pointers can also have their own pointers. These are called "pointers to pointers" or "double pointers." They are useful in situations where you need to modify the pointer itself. 
        • 
        data_type **pointer_name;

        Example

        
        int **ptr = /* some address */;  // Declaration of a double pointer
    6. Passing Pointers to Functions
      • Pointers can be passed as function arguments, allowing functions to modify the original variables or access their values indirectly. 
        • 
        void function_name(data_type *pointer_name) {
            // Function body
        }

        Example

        
        void swap(int *a, int *b) {
            int temp = *a;
            *a = *b;
            *b = temp;
        }
    Pointers in C are an essential concept for dynamic memory allocation, array manipulation, and accessing data structures. They provide low-level control over memory and enable efficient memory management and data manipulation.

    Strings in C

    In C programming, strings are sequences of characters terminated by a null character ('\0'). Strings are represented as arrays of characters and are commonly used to store and manipulate textual data. Here's an overview of strings in C
    1. String Declaration and Initialization
      • Strings in C are represented as character arrays. You can declare and initialize a string using the following syntax
        • 
        char string_name[size];

        Example

        
        char greeting[20];  // Declaration of a string with a size of 20 characters
      • Strings can also be initialized with a specific value at the time of declaration using double quotes (""):
        • 
        char string_name[] = "initial_value";

        Example

        
        char name[] = "John";  // Initialization of a string with the value "John"
    2. String Input
      • You can read a string from the user using the scanf function with the %s format specifier: 
        • 
        scanf("%s", string_name);

        Example

        
        char city[50];
        scanf("%s", city);  // Reads a string input from the user and stores it in 'city'
    3. String Output
      • You can display a string using the printf function with the %s format specifier:

        • Example

        
        char message[] = "Hello, World!";
        printf("%s", message);  // Displays the string "Hello, World!"
    4. String Functions
      • C provides a set of string functions in the header that can be used to perform various operations on strings. Some commonly used string functions include:
        • strlen: Calculates the length of a string.
        • strcpy: Copies one string to another.
        • strcat: Concatenates two strings.
        • strcmp: Compares two strings.
        • strstr: Searches for a substring within a string.
    5. String Manipulation
      • Since strings are represented as character arrays, you can access and modify individual characters within a string using array indexing.

        • Example

        
                        char name[] = "John";
name[1] = 'a';  // Modifies the second character of the string to 'a'
      • You can also use string functions like strcpy, strcat, and sprintf to manipulate strings and perform operations such as copying, concatenating, and formatting.
    6. String Comparison
      • String comparison in C is done using the strcmp function, which returns an integer value indicating the result of the comparison.
        • If the result is 0, the strings are equal.
        • If the result is negative, the first string is lexicographically smaller than the second.
        • If the result is positive, the first string is lexicographically larger than the second.

        • Example

        
    char str1[] = "apple";
    char str2[] = "banana";
    int result = strcmp(str1, str2);  // Compares str1 and str2 lexicographically

    Structure in C

    In C programming, structures (also known as structs) allow you to group together multiple variables of different data types into a single unit. Structures provide a way to create user-defined data types that can hold related information. Here's an overview of structures in C:
    1. Structure Declaration
      • To declare a structure, you define its name and list the member variables inside curly braces {}. Each member variable has a data type and a name.

        • Example

        
        struct Person {
            char name[50];
            int age;
            float height;
        };
    2. Structure Initialization:
      • You can initialize a structure at the time of declaration by providing values for its member variables inside curly braces {}.

        • Example

        
        struct Person person1 = {"John", 25, 1.75};
    3. Accessing Structure Members:
      • Structure members are accessed using the dot (.) operator, which connects the structure variable and the member name.

        • Example

        
        printf("Name: %s\n", person1.name);    // Accessing the 'name' member of 'person1'
        printf("Age: %d\n", person1.age);      // Accessing the 'age' member of 'person1'
        printf("Height: %.2f\n", person1.height);  // Accessing the 'height' member of 'person1'
    4. Nested Structures:
      • Structures can be nested inside other structures, allowing you to create more complex data structures.

        • Example

        
        struct Address {
        char street[50];
        char city[50];
        int zip_code;
        };
                    
        struct Person {
            char name[50];
            int age;
            struct Address address;
        };
    5. Passing Structures to Functions:
      • You can pass structures to functions as arguments. This can be done by passing the structure variable itself or by passing a pointer to the structure.

        • Example

        
        void printPerson(struct Person p);
        // or
        void printPerson(struct Person *p);

    File Input/Output

    File input/output (I/O) operations in C allow you to read data from files and write data to files. These operations involve opening files, reading or writing data, and closing files. Here's an overview of file I/O in C:
    1. File Handling Functions:
      • The header file in C provides functions for file I/O operations. Some commonly used functions include:
        • fopen: Opens a file and returns a file pointer.
        • fclose: Closes a file.
        • fprintf: Writes formatted data to a file.
        • fscanf: Reads formatted data from a file.
        • fgetc/getc: Reads a character from a file.
        • fputc/putc: Writes a character to a file.
        • fgets: Reads a line from a file.
        • fputs: Writes a line to a file.
        • feof: Checks if the end of the file has been reached.
    2. Opening a File:
      • To open a file for reading or writing, you use the fopen function, which returns a file pointer.

        • Example

        
        FILE *file_pointer = fopen("file_name", "mode");
        Here, "file_name" is the name of the file to open, and "mode" specifies the mode of file access, such as "r" for reading, "w" for writing (overwriting existing contents), or "a" for appending (adding to the end of the file).
    3. Closing a File:
      • After you finish reading or writing to a file, you should close it using the fclose function to release system resources. 
        • 
        fclose(file_pointer);
    4. Reading from a File:
      • You can read data from a file using functions like fscanf, fgetc, or fgets. These functions allow you to read characters, strings, or formatted data from a file into variables. 
        • 
        fscanf(file_pointer, format_specifier, variable_list);
        // or
        char ch = fgetc(file_pointer);
        // or
        char line[100];
        fgets(line, sizeof(line), file_pointer);
    5. Writing to a File:
      • To write data to a file, you can use functions like fprintf, fputc, or fputs. These functions allow you to write characters, strings, or formatted data to a file. 
        • 
        fprintf(file_pointer, format_specifier, variable_list);
        // or
        fputc(character, file_pointer);
        // or
        fputs(string, file_pointer);
    6. Error Handling:
      • It's important to check for errors during file I/O operations. Functions like fopen and fprintf return a NULL value if an error occurs. You can use this to handle errors and perform appropriate error handling.
    7. File Position:
      • The file position indicates the current location in the file where the next I/O operation will occur. Functions like fseek and ftell allow you to control and retrieve the file position.
    8. File Modes:
      • The mode argument in the fopen function specifies the file access mode. Common modes include:
        • "r": Read mode.
        • "w": Write mode (overwrites existing file or creates a new file).
        • "a": Append mode (adds data to the end of the file).
        • "rb", "wb", "ab": Binary mode for reading, writing, and appending.

    1. #include:
      • The #include directive is used to include header files in your code.

    Some Advanced Topic

    1. Bitwise Operations in C
      Bitwise operations in C allow you to manipulate individual bits of integer values. They operate on the binary representation of the values, performing logical and arithmetic operations at the bit level. Here are the commonly used bitwise operators in C:
      1. Bitwise AND (&):
        • Performs a bitwise AND operation between the corresponding bits of two operands.
        • The result is 1 only if both bits are 1; otherwise, the result is 0.

          • Example

          
        unsigned int a = 5;  // binary: 0101
        unsigned int b = 3;  // binary: 0011
        unsigned int result = a & b;  // binary: 0001 (decimal: 1)
      2. Bitwise OR (|):
        • Performs a bitwise OR operation between the corresponding bits of two operands.
        • The result is 1 if at least one of the bits is 1; otherwise, the result is 0.

          • Example

          
        unsigned int a = 5;  // binary: 0101
        unsigned int b = 3;  // binary: 0011
        unsigned int result = a | b;  // binary: 0111 (decimal: 7)
      3. Bitwise XOR (^):
        • Performs a bitwise exclusive OR (XOR) operation between the corresponding bits of two operands.
        • The result is 1 if the two bits being XORed are different; otherwise, the result is 0.

          • Example

          
        unsigned int a = 5;  // binary: 0101
        unsigned int b = 3;  // binary: 0011
        unsigned int result = a ^ b;  // binary: 0110 (decimal: 6)
      4. Bitwise NOT (~):
        • Performs a bitwise complement operation, flipping each bit of the operand.
        • The result is the binary complement of the operand.

          • Example

          
        unsigned int a = 5;  // binary: 0101
        unsigned int result = ~a;  // binary: 1010 (decimal: -6 due to two's complement representation)
      5. Left Shift (<<):
        • Shifts the bits of the left operand to the right by the number of positions specified by the right operand.
        • Zero bits are shifted in from the right, and the leftmost bits are discarded.

          Example

          
        unsigned int a = 5;  // binary: 0101
        unsigned int result = a << 2;  // binary: 10100 (decimal: 20)
      6. Right Shift (>>):
        • Shifts the bits of the left operand to the right by the number of positions specified by the right operand.
        • For unsigned values, zero bits are shifted in from the left.
        • For signed values, the sign bit is shifted in from the left, preserving the sign of the value.

          • Example

          
        unsigned int a = 20;  // binary: 10100
        unsigned int result = a >> 2;  // binary: 00101 (decimal: 5)
    2. Dynamic Memory Allocation in C
      Dynamic memory allocation in C allows you to allocate and deallocate memory dynamically at runtime. It provides flexibility in managing memory resources and is commonly used when the size of memory needed is not known at compile-time or when you want to manage memory efficiently. Here's an overview of dynamic memory allocation in C using the functions malloc, calloc, realloc, and free:
      1. malloc - Allocating Memory:
        • The malloc function is used to dynamically allocate a block of memory from the heap.
        • It takes the number of bytes to allocate as an argument and returns a pointer to the allocated memory block.
        • Syntax: void* malloc(size_t size); Syntax: void* malloc(size_t size);

          • Example

          
        int* numbers = (int*)malloc(5 * sizeof(int));
        if (numbers == NULL) {
            // Handle memory allocation failure
        }
      2. calloc - Allocating and Initializing Memory:
        • The calloc function is used to dynamically allocate a block of memory and initialize it to zero.
        • It takes the number of elements and the size of each element as arguments and returns a pointer to the allocated memory block.
        • Syntax: void* calloc(size_t num, size_t size);

          • Example

          
        int* numbers = (int*)calloc(5, sizeof(int));
        if (numbers == NULL) {
            // Handle memory allocation failure
        }
      3. realloc - Resizing Allocated Memory:
        • The realloc function is used to resize an existing dynamically allocated memory block.
        • It takes a pointer to the previously allocated memory block, the new size in bytes, and returns a pointer to the resized memory block.
        • Syntax: void* realloc(void* ptr, size_t size);

          • Example

          
        int* resizedNumbers = (int*)realloc(numbers, 10 * sizeof(int));
        if (resizedNumbers == NULL) {
            // Handle memory allocation failure or consider the original block still valid
        }
      4. free - Deallocating Memory:
        • The free function is used to deallocate a dynamically allocated memory block, returning the memory back to the system.
        • It takes a pointer to the previously allocated memory block as an argument.
        • Syntax: void free(void* ptr);

          • Example

          
        free(numbers);
